# Pacman Relaunch (3D Arcade-Spiel mit Godot)

## Beschreibung
Im Rahmen dieses Projekts wurde ein 3D-Arcade-Spiel mit der Godot Engine entwickelt, das sich am Grundprinzip von Pacman orientiert.  
Der Spieler sammelt Collectables innerhalb eines begrenzten Spielfelds und muss dabei Hindernissen sowie Gegnern ausweichen. Das Spiel ist zeitlich begrenzt und verfügt über ein Stage-System mit steigender Schwierigkeit.

## Steuerung

### Tastatur
- W / A / S / D: Bewegung
- Shift: Sprint
- Leertaste: Sprung

### Controller
- Linker Analogstick: Bewegung
- Drücken des linken Analogsticks (L3): Sprint
- Kreis-Taste (O): Sprung

## Voraussetzungen
- Godot Engine (empfohlen: Version 4.6)

## Ausführung
1. Projekt in Godot öffnen  
2. Datei `project.godot` laden  
3. Hauptszene starten
